## How to Participate

### Option 1: Comment on Existing Bills (Easiest)

1. Click the "Discuss & Comment" link on any bill
2. Read existing comments
3. Add your comment or question
4. React to comments you agree/disagree with

**No GitHub account?** Create one free at [github.com](https://github.com) (takes 2 minutes)

---

### Option 2: Propose Detailed Changes

1. Go to the [legislation repository](https://github.com/openpolicyaustralia/legislation)
2. Find the bill file (markdown format)
3. Click "Edit" (pencil icon)
4. Make your changes
5. Submit as "Pull Request"
6. Explain your reasoning
7. Community reviews and discusses

---

### Option 3: Draft New Legislation

1. Identify a structural problem
2. Draft a solution using our template
3. Submit to: [legislation repository](https://github.com/constituencyinversion/legislation)
4. Community reviews and improves it

[Full Contribution Guide](contribute.html)

## Why This Matters

### The Problem
Only well-funded institutions can draft legislation that politicians will consider. This creates structural bias: policy reflects who can afford to draft it, not what would solve problems.

### The Solution
Democratize policy capacity. If you understand a problemâ€”because you live with itâ€”you can draft a solution. No credentials required. No institutional backing needed.

### The Result
Politicians get access to high-quality, non-captured policy alternatives. The institutional monopoly weakens. They can choose based on merit, not obligation.

**This is democratic infrastructure. Use it. Fork it. Build on it.**
